# Update path & production fixes

Patch release to validate upgrades from v0.1.0 and fix production update reliability.

### Fixes

- **Worker:** Restore production entrypoint so `APP_VERSION` is set (backup jobs no longer stall).
- **Admin → System:** Live maintenance banner and update overlay during backups and applies.
- **Update apply:** Stepper progress, elapsed time, and clearer reload guidance.

### Other

- **Install:** `releases/latest/download/install.sh` for fresh installs; `update.sh` defaults to latest GitHub release.

## For operators

- **Upgrade test:** From v0.1.0 run `sudo /opt/docsops/scripts/update.sh` or use **Apply update** when the updater sidecar is configured.
- **Fresh install:** `curl -fsSL https://github.com/bjkawecki/docs-ops/releases/latest/download/install.sh | sudo bash`
