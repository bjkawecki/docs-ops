# Update test release

Patch release to verify the upgrade path from v0.1.0 (Admin update checks and `update.sh`).

### Features

- **Admin → System:** Update availability check and manual SSH upgrade instructions (no functional change beyond release tagging).

## For operators

- **Upgrade test:** From v0.1.0 run `sudo /opt/docsops/scripts/update.sh v0.1.1` after a backup, or use **Apply update** when the updater sidecar is configured.
- **Rollback:** Restore `docsops.env` and re-run `update.sh` with `v0.1.0` if needed.
